# Reflex landing page

Static landing page for [Reflex](https://github.com/Oluwaseunus/reflex), a macOS menu bar utility for smart media key routing.

The download button links to the latest GitHub Release asset in this repo. New releases are published automatically by CI in the private `reflex` repo.

## Local development

```
python3 -m http.server 8000
```

Open <http://localhost:8000>.
